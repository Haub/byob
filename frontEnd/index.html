POST

Country

https://byob-ch-mh.herokuapp.com/api/v1/countries

You can post a new country to the Database by having required parameters of dest_country && grand_total.

Demographic

https://byob-ch-mh.herokuapp.com/api/v1/demographics

You can post a new country to the Database by having required parameters of origin_country, individual_total, total_minors, dest_country_id.

PUT

Country

https://byob-ch-mh.herokuapp.com/api/v1/countries/:id

You can edit and existing entry to the Database by having required parameters of dest_country && grand_total.

Demographics

https://byob-ch-mh.herokuapp.com/api/v1/demographics/:id

You can edit and existing entry to the Database by having required parameters of origin_country, individual_total, total_minors, dest_country_id.

DELETE

Country

https://byob-ch-mh.herokuapp.com/api/v1/countries/:id

You can delete an existing entry to the Database with the country ID at the end of the url.

Demographics

https://byob-ch-mh.herokuapp.com/api/v1/demographics/:id

You can delete an existing entry to the Database with the demographic ID at the end of the url.
